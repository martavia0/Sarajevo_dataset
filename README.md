# Sarajevo_dataset
Sarajevo assets from Via et al. in prep.

- The fine_aerosol_compounds.txt file contains hourly means of the PM1 components, including sources of eBC (eBCff, eBCbb) and OA (HOA, SCOA1, SCOA2, OOA).
- The Meteo.txt contains the hourly meteorological variables data at the Sarajevo-Bjelave site.
- The Gases.txt contains the main trace gases data at the Sarajevo-Bjelave and Sarajevo-Vijnica site.
